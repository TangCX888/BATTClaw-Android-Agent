## 致谢 (Chinese)

本项目在开发过程中，受到了开源社区诸多优秀项目的启发与支持。特别感谢以下项目和技术体系为 **BATTClaw** 提供的核心驱动力：

- [Google Gemini](https://deepmind.google/technologies/gemini/) — 提供超大规模上下文与强大的多模态 AI 视觉感知能力
- [adbkit](https://github.com/nicklockwood/adbkit) — 为 Node.js 环境提供稳定可靠的 Android 调试桥接核心通信库
- [MCP Protocol](https://modelcontextprotocol.io/) — 奠定了标准化的模型上下文交互规范，使得 BATTClaw 的生态打通成为可能
- [Inquirer.js](https://github.com/SBoudrias/Inquirer.js) — 提供了极其优雅、高度可配置的 CLI 终端交互组件

*除了上述核心依赖外，我们同样感谢开源社区中无数默默奉献的极客开发者们，是你们让技术变得更有温度。*

## Acknowledgements (English)

During the development of this project, we were inspired and supported by many excellent projects in the open-source community. Special thanks to the following projects and technology stacks for providing the core driving force for **BATTClaw**:

- [Google Gemini](https://deepmind.google/technologies/gemini/) — For providing massive context scaling and powerful multi-modal AI visual perception capabilities.
- [adbkit](https://github.com/nicklockwood/adbkit) — For providing a stable and reliable Android Debug Bridge core communication library for the Node.js environment.
- [MCP Protocol](https://modelcontextprotocol.io/) — For establishing a standardized model context interaction specification, making BATTClaw's ecological integration possible.
- [Inquirer.js](https://github.com/SBoudrias/Inquirer.js) — For providing extremely elegant and highly configurable CLI terminal interactive components.

*In addition to the core dependencies mentioned above, we also thank the countless geek developers silently contributing in the open-source community. It is you who make technology warmer.*
